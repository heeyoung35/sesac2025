마우스와 키보드 이벤트

1. 마우스 이벤트

    사용자의 마우스 움직임에 반응합니다.
    `click`(클릭), `dblclick`(더블클릭), `mousedown`(누를때), `mouseup`(뗄때), `mousemove`(움직일때) 등이 있습니다.

2. 키보드 이벤트

    사용자의 키보드 입력에 반응합니다.
    - keydown: 키를 누르는 순간 (누르고 있는 동안 계속 발생)
    - keyup: 키에서 손을 떼는 순간
    - event.key: 입력된 문자를 알려줌 (ex: 'a', 'A', 'Enter')
    - event.code: 실제 물리적인 키 코드를 알려줌 (ex: 'KeyA', 'Enter')
